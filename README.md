## Run main file (default output without `name` argument)
```shell
go run main.go

Hello there, world!
```
## Run main file (default with `name` arg)
```
go run main.go -name="Conor Heffron"

Hello there, Conor Heffron!
```
## OR Build & Run Package `go-world`
```shell
go build -o go-world .

./go-world -name="Conor Heffron" 
```
